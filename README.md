# BookStorage

The databases lie on the MLab and all what you need is start server and run gulp 

## Server

### Installation and usage
In dir `server` Install the npm:
```bash
$ npm install
```
and then Start the server:
```bash
$ node server.js
```
## Client

### Installation and usage

Install the npm in root dir:
```bash
$ npm install
```

and start gulp:
